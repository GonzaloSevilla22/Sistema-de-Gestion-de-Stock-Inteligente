Permite métricas y dashboard.

Permite demostrar uso de IA agregando sugerencias automáticas de reposición.

5. IA (Puntos Fuertes para el TP)

Acá ganás la parte de "Desarrollo asistido por IA".

Endpoint:

POST /ai/recommendation

La IA recibe:

{
  "producto": "Teclado Logitech",
  "stock_actual": 3,
  "stock_minimo": 10
}

Y devuelve:

Se recomienda reponer el producto dentro de las próximas 48 horas.

Claude
o
Gemini API Free

No hace falta algo complejo.
